**❤️ Heart Disease Prediction System**

An AI-powered web application that predicts the likelihood of heart disease based on user health parameters. This project leverages machine learning to provide quick, data-driven insights for early detection and awareness.



🚀 Live Demo

🔗 https://heart-disease-prediction-mocha.vercel.app/



📌 Overview

This project is designed to help users assess their risk of heart disease using a trained machine learning model. By analyzing various medical attributes such as age, cholesterol, blood pressure, and lifestyle factors, the system provides instant predictions and insights.



✨ Features

🧠 AI-Based Prediction – Uses machine learning to evaluate heart disease risk
📊 Instant Results – Get predictions in real-time
📋 User-Friendly Interface – Simple and clean UI for easy data input
🔍 Health Insights – Understand contributing risk factors
⚡ Fast & Responsive – Deployed on Vercel for high performance



🛠️ Tech Stack

Frontend:
HTML, CSS

Backend:
Python / FastAPI 

Machine Learning:
Scikit-learn
Models: Logistic Regression, Random Forest 



📊 How It Works

Input Data
User enters health parameters like age, cholesterol, BP, etc.

Model Processing
Machine learning model analyzes the input data.

Prediction Output
System returns:
Risk level (Low / High)
Prediction result (0 or 1)

⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/your-username/heart-disease-prediction.git
cd heart-disease-prediction
2. Install dependencies
pip install -r requirements.txt
3. Run backend
python app.py
4. Run frontend
npm install
npm start
